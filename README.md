# Projet-TaskMaster
Projet 1 de passerelle

__OBJECTIF__ : Créer un assistant qui gère tes rappels et tes tâches directement dans un salon Discord. (To do list discord)

__Cahier des charges__ : 
1. Commandes de bases :
      - !add [name] : Enregistre une nouvelle tache
      - !list : Affiche l'ensemble des taches à réaliser
      - !done [id] : Marque la tache d'id "id" comme terminée
2. Les tâches ne doivent pas s'effacer lorsque le bot redémare
3. Message automatique à une heure précise définie par l'utilisateur.
4. Une commande !stat pour afficher les taches réalisée

__Technologies__ :
Python 3.10+
Bibliothèque discord.py
Base de données : SQLite
